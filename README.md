# EduPath — Intelligent College Mentoring Platform

![EduPath Header](https://images.unsplash.com/photo-1523240715634-19069486a69d?auto=format&fit=crop&q=80&w=2000)

EduPath is a state-of-the-art college mentoring and academic tracking platform designed to move beyond traditional grading. By utilizing **Interactive Knowledge Graphs**, **AI-Powered Interventions**, and **Multi-Role Dashboards**, EduPath provides a 360-degree view of student progress.

---

## 🚀 Key Features

### 🎓 For Students
- **Interactive Knowledge Graph:** Visualize subjects as a network. See prerequisites, unlocks, and track mastery in real-time using D3.js.
- **AI Study Assistant:** A context-aware chatbot that provides personalized study plans, generates quizzes, and creates flashcards for any topic.
- **Coding Log:** Track daily problem-solving streaks (LeetCode/CodeChef) and visualize consistency.
- **Career Readiness:** Guided checklists for industry-specific tracks (Data Science, Full-Stack, etc.).
- **Attendance Heatmap:** Beautiful calendar visualization of daily attendance with threshold warnings.
- **Weekly Pulse:** Simple 1-click emotional check-in for mentors to track student well-being.

### 👨‍🏫 For Teachers (Mentors)
- **Risk-Sorted Roster:** Automatically identifies students who need help based on mastery dips, attendance drops, and pulse checks.
- **Mentoring Suite:** AI-powered session summarization. Paste session notes to generate summaries and student action items automatically.
- **Class Mastery Heatmap:** Grid-view of the entire class's progress across all subjects.
- **Markdown Notes Editor:** Create and edit rich-text study notes that are instantly available to all students on their Knowledge Graph.

### 🏛️ For Admins
- **Global Control Room:** Manage 8+ engineering departments, academic semesters, and global announcements.
- **Semester Manager (Time Machine):** Create and archive semesters. The "Time Machine" feature allows users to toggle back and view mastery data from past years.
- **User Management:** Full CRUD for students, teachers, and parents.

### 👪 For Parents
- **AI Learning Stories:** Human-friendly weekly progress summaries generated by AI.
- **Child Progress Overview:** Read-only access to child's CGPA, attendance, and upcoming deadlines.

## 🛠️ Tech Stack

- **Frontend:** React 18, Vite, TypeScript
- **Styling:** Tailwind CSS (Modern Editorial Theme), Shadcn UI
- **Data Visualization:** D3.js (Force-directed Knowledge Graphs), Recharts
- **Backend:** Supabase (Auth, PostgreSQL, Real-time)
- **AI Engine:** Google Gemini via Supabase Edge Functions
- **Notifications:** Resend API for automated email reporting
- **Offline:** PWA (Progressive Web App) support via `vite-plugin-pwa`
- **Exports:** PDF Report generation via `html2pdf.js`

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nikhileshhh2603/edupath-navigator-main.git
   cd edupath-navigator-main
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Variables:**
   Create a `.env` file or set up your Supabase project:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

4. **Run Locally:**
   ```bash
   npm run dev
   ```

5. **Deploy Edge Functions:**
   ```bash
   supabase functions deploy ai-tutor
   supabase secrets set LOVABLE_API_KEY=your_key
   supabase secrets set RESEND_API_KEY=your_key
   ```
